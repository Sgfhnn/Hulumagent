# Hulumagent

**Ethiopia's first multilingual AI agent platform for businesses.**

Live: [hulumagent.online](https://hulumagent.online)

---

## Overview

HulumAgent provides businesses with intelligent AI agents that automate customer support, boost engagement, and operate 24/7 across platforms including Telegram, WhatsApp, and Web.

Built specifically for the Ethiopian and African market with full support for Amharic, Oromic, and English — making it the first AI agent provider in Ethiopia serving local languages at this scale.

---


## Services

**AI Customer Service Chatbots**
Handles FAQs, order tracking, and customer support 24/7. Built for e-commerce, banks, and educational platforms.

**WhatsApp & Telegram AI Agents**
Conversational agents that handle support, sales, and follow-ups directly where customers already communicate.

**Lead Generator AI Agent**
Qualifies users, collects contact details, and syncs with Facebook, Instagram, and websites automatically.

**Survey & Feedback AI Agent**
Collects multilingual user feedback with sentiment analysis and exportable reports. Built for NGOs, research, and product teams.

**Voice AI Assistants**
Voice-enabled bots supporting Amharic, English, and Oromic speech recognition and text-to-speech.

**Live Chat + AI Hybrid Support**
Combines AI agents with human handover logic for seamless 24/7 customer support at scale.

## Tech Stack

**Frontend:** React, Next.js, TypeScript, Tailwind CSS
**Backend:** Node.js, Express, Python
**AI:** Claude API, OpenAI API, Gemini API, NLP
**Database:** PostgreSQL, Supabase, MongoDB
**Platforms:** Telegram Bot API, WhatsApp Business API, Web
**Languages Supported:** English, Amharic, Oromic

---

## Key Features

- Multilingual support (English, Amharic, Oromic)
- Voice input and text-to-speech in local languages
- Multi-platform deployment (Web, WhatsApp, Telegram)
- Live chat with AI handover logic
- Custom built agents for any business need
- Sentiment analysis and reporting

---

## What Makes It Different

HulumAgent is not a generic chatbot platform. It is built specifically for Ethiopian and African businesses, understanding local languages, local constraints, and local customer behavior. Every agent is custom built — no templates, no generic scripts.
